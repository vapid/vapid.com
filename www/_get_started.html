Get started

If you’re interested in kicking the tires, are comfortable with dev environments, and are just excited to check Vapid out, then please visit the open source repo on GitHub to download Vapid and get started.

This is a great way to get familiar with the app, and prep your site(s) for deploying on the Vapid hosted service.

If you’ve got an interesting project and are ready to deploy on the Vapid platform, drop us a message at hello@vapid.com.

npm install -g vapid-cli
vapid new path/to/project/folder
cd path/to/project/folder
vapid start
